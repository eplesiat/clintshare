# CLINTshare

Software to share CLINT data

## Dependencies
- python>=3.9.13
- numpy>=1.21.5
- PyYAML>=6.0
- tqdm>=4.66.1

## Installation

`CLINTshare` can be installed using `pip` in the current directory:
```bash
pip install .
```

## Usage

```
clintshare <path-to-data-directory>
```
